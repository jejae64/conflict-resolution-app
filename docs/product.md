# Workplace Conflict Resolution Program

## 1. Product Goal

This program helps users:

- regulate emotions
- structure conflict situations
- prepare better communication

It does NOT judge who is right or wrong.

---

## 2. Target Users

- office workers
- people experiencing conflict with managers or coworkers
- people who struggle to express themselves properly in conflict situations

---

## 3. Core Features (MVP only)

1. conflict situation input
2. emotion detection and intensity
3. underlying need analysis
4. conflict type classification
5. constructive dialogue + action guidance

## 5. Non-goals (NOT included in MVP)

- login / authentication
- payment system
- admin dashboard
- advanced AI tuning
- multi-user features


## Safety Note

This program does not handle:

- violence or abuse situations
- harassment or threats
- severe psychological distress

In such cases, users should seek professional help.

It does NOT judge who is right or wrong.
It focuses on helping users understand and communicate better.

---

## 6. Success Criteria

- user understands their situation more clearly
- user can use the suggested dialogue in real conversation
- user feels emotionally calmer after using the tool
